いらっしゃいませませ〜🐌
るぅちゃんこと、四条月です🌘
　
本日は、ツアーでいただいた
お花のお礼を伝えさせてください！
もっともっと早くに伝えるべきだったのに、
今更でごめんね(´・ω・)
　
本当に本当にありがとうございます！！！
　
私がアイドルになって、初めてファンの方々からいただいたお花です😌
一生忘れません！！！
さあっ、自慢してくぞ〜💫(｀・ω・´)ﾄﾞﾔ
　
［目次］
・ぴかぴか〜⭐️
・きらきら〜⭐️
・おるな様🌕
・総括🐌？
　
【ぴかぴか〜⭐️】🦉🦉🦉🦉🦉🦉🦉🦉
　

　
真ん中の黄色のお花が、お月様で、私のことを表しているんです！
他にも秋元康先生プロデュースの坂道さんを意識している装飾だったり、、、
デザインひとつひとつに、沢山のストーリーが詰まってるんです！！（ ;  ; ）
　
詳しくは、2人の秘密です😉🌙
　
【きらきら〜⭐️】🦉🦉🦉🦉🦉🦉🦉🦉
ライブ当日のステージ裏。
　
事前にお花送るねーって伝えてくださった方もいらっしゃったので、
お花を探すために珍しく私が俊敏な動きをしていました🐌🌪️笑笑
　
だってフラスタ解禁されてから、
皆さん唐突に好きなお花を尋ねてくるんだもん！！！笑笑
んふふってなるの堪えてました(｡-_-｡)
　
でも事前にお花送るねーって伝えずに、
サプライズで送ってくださる方もいまして！
送ってくださった方々との思い出を振り返っては、
サプライズというかポーカーフェイスが上手だなあって😎
にやにやしてしまいました( ´ ▽ ` )
　
きらきら〜⭐️なお花のおかげで、
真っ暗闇も怖くないです！
いつも喜ばしてくれてありがとう( ´ ▽ ` )
　

　
【おるな様🌕】🦉🦉🦉🦉🦉🦉🦉🦉
かーわーいーいー（ ;  ; ）
　

　
・葉っぱのリースが円を描いてるお花🌱
お月様を観るぅちゃんしてるじゃないですかー！！！(ﾟ∀ﾟ)
しかもナナパレのフラッグパフォーマンス用の衣装着てるぅ( ´ ▽ ` )/🚩
嬉し過ぎて私も望遠鏡買っちゃおうかな？！🔭
いつも可愛いを作ってくれてありがとう🐇🏠🫶🏻
私もう大ファンですう（ ;  ; ）
　
・ひまわりとネモフィラのバスケット🧺🌻
私の好きなお花達(｀・ω・´)✨
ツツジも好きって伝えたけど、流石にツツジは入ってなかった😂
好きなお花聞かれて、よくツツジ吸ってたよって伝えたのwww
　
今の季節じゃないお花も咲いてて、びっくり嬉しいです（ ;  ; ）
多分そっくりさんなお花かな？凄い！！
ナナニジ入ってから忙しくて、向日葵畑もネモフィラ畑も観に行けてなかったから嬉しかったなあ😌幸せだなあ。
　
私小さい頃、
ひまわりの髪飾りを頭につけてたんです😂🌻
お花の写真見せたら四条の母上も喜んでました( ´ ▽ ` )/🌻
　
向日葵でお月様を作ってるのも気づいたよ！
お月様の骨組みがバスケットの中に隠れてて、なんだかほっこりしました(*´-`)
　

　
お花２つとも沢山考えてくれて送ってくれてありがとう( ´ ▽ ` )
　
私もモノづくりが好きだからね、
メンバーから、
ファンの皆さんもるぅちゃんに似て芸術のセンスがピカピカだねーって言われちゃった(//∇//)
　
【総括🐌？】🦉🦉🦉🦉🦉🦉🦉🦉
素敵なお花達を本当にありがとうございます！
だけど皆様、ご無理のない範囲で！（ ;  ; ）👍🏻✈️🚄
会いに来てくれるだけで、気持ちだけで本当に嬉しいです(｀・ω・´)🎶
　
その他にも新メンバー宛てのお花を見つけて、
私達のことを受け入れ、
応援してくださる方々の心の広さと温かさを
改めて感じました！
本当に本当にありがとうございます💐
　

　
ご卒業されたメンバーを元々応援していた方で、
特典会で私に会いに来てくださり、
複雑なお気持ちを伝えてくださる方もいらっしゃるんです。
　
伝えてくれてありがとう😌
いつも焦ってしまって「ありがとう」しか伝えられてない気がするから🤔、
改めて、
本当に、
伝えてくれてありがとう。
　
でも、皆さん優しいから
その複雑なお気持ちを伝えてくださった後に、
「るぅちゃん頑張れ！」
「るぅちゃん期待してるよ！」
「応援したいってまた思えたんだ！」
って、前向きな言葉を伝えて、
笑顔で特典会のお話を締めくくってくださるんです。
　
ちゃんと私はその寂しさに寄り添えてるかな？
また一緒にアイドルのキラキラした所を見れてるかな？
あ、私の場合キラキラじゃなくてぴかぴか〜⭐️かな？笑笑
　
以前かなえさんに、
会ったことないのにご卒業されたメンバーにもリスペクトできるのが偉い？凄い？よ！
会ったことないのにどうしてー( ´ ▽ ` )？？
　
って尋ねられたことがあるんです。
私、理由なんて考えたことなくて！笑笑Σ（・□・；）ｵﾛｵﾛ
　
アイドルに憧れていたのは、もちろんリスペクトの理由のひとつだなって。
選ばれた人しかできないから！
　
でもナナニジで活動する中で、
「続ける」ってすごいなってリスペクトも生まれました。
「こんなの続けられるか！」ってナナニジを批判している訳ではなくて笑笑、
「好きを続ける」ことへのリスペクトです！
　
私がアイドルに惹かれた理由のひとつが、
必死さです。
世間相手に普通の女の子だった人間が立ち向かっていく姿がカッコ良いなって！
　
総選挙とか、公式ライバルとか、
本気で挑んでる必死がカッコ良くて惹かれたんです。
　
どんでん返しや、シンデレラストーリーも、
アイドルらしくて面白くて憧れますね。
　
まあもちろん、可愛い子が好きっていうのもかなりあるんですけどね( ´ ▽ ` )✌️ﾁｮﾛﾁｮﾛﾋﾟｰｽ
　
11枚目シングル「僕は今夜、出て行く」のサビの歌詞。
「明日を信じてごらん」
という言葉が好きです。
「ごらん」って良いですよね、やさしい世界🐌🌍
　
まだまだ好きな歌詞はあるのですが、
リリースがまだなのでこの辺りで(*´-`)ｽｯ
　
四条月というアイドルを、
こんな私の"明日を信じてみよっかな"、
って思っていただけるよう
これからも努めてまいります🦉🌙
　
アイドル人生楽しむぞ〜！！！
　
p.s.🦉🦉🦉🦉🦉🦉🦉🦉
お花にささってた旗の裏に、マゼンタって書いてあった😂😂
　

　
お花屋さんへ。
分かりづらいメンカラでごめんなさい笑笑
るぅより。
　
今度からお花屋さんには「柴漬け色」って頼んでな👍🏻www
　
p.s.🦉🦉🦉🦉🦉🦉🦉🐌
超会議2023で行われた「超アイドル祭」のライブのアーカイブ観たらね、
すごーく私よかった！！！😂笑笑
自分で言って恥ずかしいんだけどもww、本当に四条良かったの！！！(● ˃̶͈̀ロ˂̶͈́)੭ꠥ⁾⁾
　
ステージライトを味方にしてて、キラキラしてて、思い描いてたアイドルにすごーーーく近かった。
観ながらなんかうるっと来ちゃいましたもん！
　
5/29(月)までアーカイブあるって！((ちゃっかり宣伝😂🛒
是非行けなかったよって方観てみてね〜⭐️
　
　
　
　
　
ありがとうございました
また来てね〜🦉🌙
欢迎光临、欢迎光临〜🐌
我是大家熟知的 Ruu-chan，也就是四条月🌘
　
今天，请让我来表达对巡演中收到的
花篮的谢意！
明明应该更早一点告诉大家的，
到现在才发出来真是抱歉呀(´・ω・)
　
真的真的非常感谢大家！！！
　
这是我成为偶像后，第一次从粉丝们那里收到花篮😌
我一生都不会忘记的！！！
那么，我要开始炫耀喽〜💫(｀・ω・´)嘿嘿
　
［目录］
・闪亮亮〜⭐️
・亮晶晶〜⭐️
・O-Luna大人🌕
・总结🐌？
　
【闪亮亮〜⭐️】🦉🦉🦉🦉🦉🦉🦉🦉
　

　
中间那朵黄色的花代表着月亮，也就是我！
此外还有意识到了秋元康老师制作的坂道系列的装饰等等……
每一个设计细节里，都充满了许多故事！！（ ; ; ）
　
详细的内容，是属于我们两人的秘密哦😉🌙
　
【亮晶晶〜⭐️】🦉🦉🦉🦉🦉🦉🦉🦉
演唱会当天的后台。
　
因为也有人事先跟我说过“会送花给你哦ー”，
为了寻找花篮，我难得地展现出了敏捷的动作🐌🌪️笑笑
　
毕竟自从祝花解禁之后，
大家都会突然跑来问我喜欢什么样的花嘛！！！笑笑
我当时一直忍着不让自己发出“唔呼呼”的笑声(｡-_-｡)
　
不过也有人事先没说要送花，
而是给了我一个惊喜！
回顾与送花各位的回忆，
比起惊喜，感觉大家更擅长摆出一副扑克脸呢😎
让我忍不住在一旁偷笑( ´ ▽ ` )
　
多亏了这些亮晶晶〜⭐️的花朵，
就算在漆黑的暗处也不会害怕了！
谢谢你们总是让我这么开心( ´ ▽ ` )
　

　
【O-Luna大人🌕】🦉🦉🦉🦉🦉🦉🦉🦉
好——可——爱——（ ; ; ）
　

　
・叶子花环绘成圆圈的花篮🌱
这不就是在“观赏Luna”嘛——！！！(ﾟ∀ﾟ)
而且还穿着 Nanapare 旗帜表演用的服装( ´ ▽ ` )/🚩
真是太开心了，我是不是也要去买个望远镜呢？！🔭
谢谢你总是为我打造这份可爱🐇🏠🫶🏻
我已经成了你的超级粉丝啦（ ; ; ）
　
・向日葵和粉蝶花的提篮🧺🌻
全是我喜欢的花(｀・ω・´)✨
虽然我也说过喜欢杜鹃花，但果然杜鹃还是没放进去呢😂
被问到喜欢的花时，我回答说小时候经常吸杜鹃花蜜www
　
能看到在这个季节并不会盛开的花朵，真是又惊讶又开心（ ; ; ）
大概是长得很像的花吧？好厉害！！
加入 Nanani-ji 之后因为太忙，向日葵田和粉蝶花田都没能去看成，所以真的好开心呀😌好幸福。
　
我小时候，
经常在头上戴向日葵发饰呢😂🌻
把花的照片给四条的母亲看，她也很开心( ´ ▽ ` )/🌻
　
我也注意到用向日葵做成了月亮的形状哦！
月亮的骨架就藏在提篮里，总觉得心里暖洋洋的(*´-`)
　

　
谢谢你为这两份花篮费了这么多心思送给我( ´ ▽ ` )
　
因为我也很喜欢手工制作，
被成员们说：
“粉丝们也和 Luna-chan 很像，艺术感都闪闪发光的呢ー”(//∇//)
　
【总结🐌？】🦉🦉🦉🦉🦉🦉🦉🦉
真的非常感谢这些美妙的花篮！
但是各位，请在不勉强自己的范围内支持哦！（ ; ; ）👍🏻✈️🚄
只要能来看我，光是这份心意我就已经非常开心了(｀・ω・´)🎶
　
此外，看到送给新成员们的花篮，
再次感受到了
接纳并支持我们的大家的宽广心胸和温暖！
真的真的非常感谢💐
　

　
有些原本支持已毕业成员的粉丝，
会在特典会上来见我，
向我倾诉他们复杂的心情。
　
谢谢你告诉我😌
总觉得我每次都因为太慌张，只能说出“谢谢”🤔，
所以我想再次说一次，
真的，
谢谢你愿意告诉我。
　
但是，因为大家都太温柔了，
在传达了那份复杂的心情后，
还会对我说“Luna-chan 加油！”
“Luna-chan，我很期待你哦！”
“我又产生了想要支持的心情了！”
像这样说些积极的话语，
微笑着结束特典会的话题。
　
我有没有好好地体贴那份寂寞呢？
有没有和大家一起看到偶像闪闪发光的地方呢？
啊，对我来说可能不是亮晶晶，而是闪亮亮〜⭐️吧？笑笑
　
以前萌前辈曾对我问过，
“明明没见过面，却能对已经毕业的成员保持敬意，真的好了不起/好厉害呀！”
“明明没见过面，为什么能做到呢( ´ ▽ ` )？？”
　
我当时完全没想过理由！笑笑Σ（・□・；）慌乱
　
对偶像的憧憬，当然是致敬的理由之一。
因为那是只有被选中的人才能做到的事情！
　
但在 Nanani-ji 的活动中，
我也产生了“坚持下去”真的很了不起的敬意。
这并不是在批判 Nanani-ji 说“这种事怎么坚持得下去！”啦笑笑，
而是对“坚持热爱”这份心意的敬意！
　
我被偶像吸引的理由之一，
是那份拼命感。
原本只是普通女孩子的普通人，挺身面向世界的姿态真的很帅气！
　
像是总选举，或者是官方对手，
那种全身心投入其中的拼命感很帅，深深吸引了我。
　
逆风翻盘，或者是灰姑娘的故事，
这种很有偶像感的有趣桥段我也很憧憬呢。
　
嘛，当然，我超级喜欢可爱的女孩子也是很大一部分原因啦( ´ ▽ ` )✌️随性剪刀手
　
第11张单曲《今晚，我将启程》副歌的歌词。
我很喜欢“试着相信明天吧”这句话。
“试着（去看吧）”这种说法真好呢，温柔的世界🐌🌍
　
虽然还有很多喜欢的歌词，
但因为还没发售，所以就先聊到这里吧(*´-`)收
　
为了能让大家觉得，
因为四条月这个偶像，“试着相信一下我的明天吧”，
今后我也会继续努力的🦉🌙
　
我要享受偶像人生啦〜！！！
　
p.s.🦉🦉🦉🦉🦉🦉🦉🦉

Luna上。

我表现得超级好！！！😂笑笑
虽然自己说有点不好意思ww，但四条真的表现得很棒！！！(● ˃̶͈̀ロ˂̶͈́)੭ꠥ⁾⁾
　
舞台灯光都站在我这一边，亮晶晶的，非常接近我理想中的偶像样子。
看着看着甚至感觉眼眶有点湿润了呢！
