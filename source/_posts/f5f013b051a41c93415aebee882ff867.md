<br/>
<br/>
<br/>
这一次演唱会的进行方式，应该让不少人感到惊讶吧？
<br/>
<br/>
‐通过咖啡厅“SEVENS RAINBOW”从开业到打烊的全过程来展现一场演唱会‐
<br/>
<br/>
在关于周年庆（Anira）的会议上，听到要用这种方式进行演唱会的时候，我完全想象不出会变成什么样的表演呢。
<br/>
演技环节给我的感觉真的就像在演舞台剧一样。老师告诉我们，没有台词时的动作才是最重要的。于是成员们聚在一起讨论，“如果是咖啡厅店员的话应该会这样动吧”，“如果是这个成员在咖啡厅打工的话应该是这种感觉吧”之类的，就这样一点点地完成了排练。
<br/>
就像平时记舞蹈站位一样，要记住说到这句台词时去这里、再去那里……
<br/>
在反复进行整场彩排的过程中，自然而然地就变成了“这时候去吃奈央酱做的蛋糕吧—”或者“和和前辈一起睡午觉吧—”之类的，就像是麻丘真央在饰演麻丘真央会做的事一样！🧚‍♂️
<br/>
既是我，又不像是我，感觉非常奇妙呢。。。
<br/>
<br/>
起初还在摸索阶段，心里确实有点不安，但现在圆满结束后回想起来，觉得我们发掘出了 22/7 新的演唱会形式呢💍
