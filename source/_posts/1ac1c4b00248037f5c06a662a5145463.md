首次写博客。<br/>
<br/>
我是新成员麻丘真央。大家能看到这篇博客，说明巡演的第一天已经顺利结束了吧！<br/>
现在我正思考着明天即将到来的演唱会。博客的后续我想在明天写，但还是想把此刻还未体验过演唱会的麻丘真央在演出前一天的心情记录在这里。<br/>
顺便说一下，我现在还不紧张。不过等到了观众面前，一定会涌现出各种情感，肯定会紧张的，大脑大概也会变得一片空白，连动作都会忘掉吧。但是，从通过试镜决定成为22/7成员的那天起，我就一直在认真面对22/7。哪怕是不习惯的事情堆积如山、异常辛苦的课程和排练，我也在前辈和成员们的帮助下克服了过来，所以我相信一定没问题的。<br/>
前辈们给了我“为了享受演唱会，排练时要全力以赴”的建议。这句话真的深深地印在我的心里......如果能展现出成果的话就太开心了!!<br/>
<br/>
我有着从击剑运动员转型为声优偶像这样稍微有点特别的经历，我想应该有很多人会觉得“为什么？”吧。关于这件事说起来就长了，希望以后能在博客里认真地跟大家聊聊。<br/>
<br/>
在与曾经作为运动员的环境大不相同的22/7世界里，能挑战从小就想尝试的唱歌跳舞，每天真的都过得既开心又新鲜。<br/>
边跳边唱比我预想中要难呢笑 再次对偶像们肃然起敬！<br/>
<br/>
而且我时常感觉到，能加入这个组合真是太好了。因为有着优秀的成员们以及工作人员们的支持，我才能安心地进行活动。<br/>
虽然这只是我个人的偏见，但我以前一直觉得偶像成员之间的关系会很僵。但在进入22/7之后，这种偏见就烟消云散了。每天发现前辈们和新成员们的新的一面时，我都会觉得大家真的是很好、很出色的人，心情就会变成“好喜欢大家呀”。能拥有这种心情真的很幸福呢( ¨̮ )<br/>
而且我真的觉得自己遇到了很好的人们，非常感谢现在的环境。正因如此，我必须更加更加努力才行！！！！<br/>
这就是我演唱会前一天的想法笑<br/>
<br/>
怎么样？演唱会成功了吗～？明天的麻丘酱！！笑<br/>
<br/>
早上好！现在正坐在前往会场的电车里。虽然还没有紧张，但胸口深处已经开始有种揪紧的感觉了。而且还兴奋得不由自主地嘴角上扬。今天无论发生什么事我都不想哭……虽然我觉得应该不会因为难过而哭，但我经常会流下喜悦的泪水，所以我希望能把那份眼泪憋到最后最后和大家告别的时候。但如果从overture开始就哭了的话，那对不起啦。<br/>
<br/>
<br/>
现在演唱会结束了！！！<br/>
<br/>
真的真的非常开心！！overture的视频也制作得非常棒，而且第一次看到坐满观众的会场时，荧光棒的景色实在太美了，让我感动得直起鸡皮疙瘩。<br/>
虽然我以为会很紧张，但意外地并没有！比起紧张，更多的是“啊，我真的成为了22/7的一员了啊”这种实感，感觉真的非常幸福。谢谢大家让我看到了那么美的荧光棒景色！如果不开始这份活动的话，我绝对无法看到那样美丽的景色。真的非常开心。<br/>
而且对于还名不见经传的我，在MC时会场大家的反应非常温柔，让我很感动。也谢谢大家送我的那么多掌声！<br/>
明明想着无论发生什么事都不想哭的，结果在前半部分的第3首歌左右，因为一开始唱得太用力导致嗓子哑了，觉得自己很不争气，就在MC的时候哭了出来……好不甘心。嗓子哑了让我觉得自己专业意识不足，很不甘心，而对于因此哭出来的自己也感到非常非常不甘心。今后为了能在顾及嗓子的同时游刃有余地完成演出，我会继续进行更多更多的练习！<br/>
下一场演出绝对想不流泪地结束。<br/>
<br/>

<br/>
<br/>
读这篇博客的大家一定都很喜欢22/7吧。我也最喜欢22/7了。从今以后，我既是组合的一名粉丝，也是一名成员。为了能给最喜欢的22/7出一份力，我会拼尽全力努力下去的！<br/>
而且为了能成为偶像界的运动员“麻丘真央”，我会加油的。请多多支持！<br/>
虽然写得有点长，但非常感谢大家能读到最后！今后也请多多关照！<br/>
