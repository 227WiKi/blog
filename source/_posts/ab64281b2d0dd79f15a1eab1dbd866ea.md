冻疮<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
是我在冬天最害怕的东西。<br/>
<br/>
<br/>
<br/>
<br/>
要说为什么突然写这个呢，<br/>
<br/>
<br/>
<br/>
<br/>
是因为我忘带手套了……<br/>
<br/>
<br/>
<br/>
<br/>
是的，陷入了大危机。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
虽然在东京的时候还没事，但这里的空气好刺痛啊<br/>
<br/>
<br/>
<br/>
<br/>
以前住在这里的时候，手总是被说像冰块一样冷<br/>
<br/>
<br/>
一旦长了冻疮，手就会肿到原来的1.5倍左右<br/>
连握笔都会痛呢<br/>
<br/>
<br/>
<br/>
<br/>
绝不能就这样长冻疮呀っ<br/>
<br/>
<br/>
<br/>
于是我想了想。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
最后钻研出的方法。<br/>
<br/>
<br/>
<br/>
首先，右手要用力握紧，把大拇指藏在里面<br/>

<br/>
<br/>
然后为了尽量不吹到风，固定住不要乱动。<br/>

<br/>

<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
最后，使劲地低下头<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
这样就完美啦！<br/>
<br/>
<br/>
<br/>
<br/>
但是这个姿势会让视野变得非常狭窄，所以必须时刻注意电线杆才行<br/>
<br/>
<br/>
<br/>
还挺惊险刺激的<br/>
<br/>
<br/>
结果就是走路的速度变慢了。<br/>
