4月20日，我迎来了桐生塔子酱这位在22/7生涯中无可替代的搭档。
<br/>
当舞台上投射出角色形象时，其他新成员都很开心的样子，唯独我没法打心底里高兴起来。
<br/>
当然，开心肯定是开心的。只是没想到这么快就能获得属于自己的角色……。
<br/>
但是，觉得自己声音没什么特色，也不知道能不能演好戏，尽往不好的方面想了……。
这么出色的角色交给我来担任真的好吗，如果让别人来演塔子酱，她会不会变成一个被更多人喜爱的孩子呢……我一直在这样想。
<br/>
前辈们在创造了各种历史的过程中与角色并肩前行，而什么都不会的我真的可以拥有角色吗，我也曾这样思考过。
<br/>
就在那时，萌前辈对所有新成员说：“大家能拥有角色我真的好开心呀，接下来也要享受和角色们并肩作战的过程哦——”。萌前辈的这一句话真的深深救赎了我。
<br/>
<br/>
<br/>
虽然现在也还有不自信的地方，但塔子酱是只有我才能演绎的，而且也没有标准答案，所以我就决定啦，“要建立起自己心中世界第一可爱的塔子酱形象！！”🫶！！！
<br/>

遇到各种烦恼时，总能帮到我的就是成员们。

22/7真的是个非常棒的团体🥲🥲🥲
