tiktokバズるぅ🌙

22/7 WiKi2024-01-172026-02-04

✨ 由 Google Gemini 提供翻译 · 仅供参考

﻿いらっしゃいませ、
四条どすえ( ´ ▽ ` )

















【おしながき】
⚪︎結果が数字でコンニチハ
⚪︎説得力がナッシング
⚪︎歌番組呼ばれなかったヒグチカッター
⚪︎最後に











⚪︎結果が数字でコンニチハ
tiktokの個人アカウントにて、嬉しいことがありました。。。
なんと！！！
あるダンス動画が、30万回再生を記録しました！！！

👏🏻👏🏻👏🏻👏🏻👏🏻👏🏻👏🏻👏🏻（ ; ; ）
いわゆる、現代用語の「バズる」が起きたのです。









↑我らが所属事務所、バズウェーブのバズです😎✌🏻

もうバズるたびに通知が鳴り止まなくて、
手が震えてしまう四条です。(´・ω・)ｺﾜｲ










実はそれ以前に2023年4月、
個人アカウントを開設して4本目の動画でも
ダンス動画がバズりました。

その動画はもうすぐ15万回再生に達しそうです🎉








この4本目のtiktokの再生回数が伸びた際、
嬉しかったけれど私は全く浮かれてなくて。

次の「結果」を出さねば意味がないと
真剣さゆえに怖い顔をしていたかも知れません。笑笑









そしたら前回バズった4本目の動画の
2倍以上の、
30万回再生越えの
バズりを果たす事ができました。。。！











この結果を掴むことができて、
心底嬉しかったです。
そして少し肩の力を抜くことができました( ´ ▽ ` )









肩に力が入っていた理由としては、

私が、数字で結果が欲しいこと。
動画一本を撮影するのに苦労すること。
自分からtiktokを始めたいと言い続けてたこと。

それらの考えが頭をぐるぐるとしていました。☁️













⚪︎説得力がナッシング

自分からtiktokを始めたいと言い続けてたこと。。。

実はずっとずっと、私が演じる一之瀬蛍ちゃんで
tiktokをやりたいとお願いしていました。











でもまあ蛍ちゃんが動くのは、
当たり前ですが私が動くのより難しいので。

まずは私が結果を出さないと、難しいのです。









やりたいって言うのは簡単だけど、
昔の私には説得力がありませんでした。

それはそう( ＾∀＾)新人。落ち込んでいられなーい♪









tiktokのために本業であるライブパフォーマンスを早めに仕上げたり、
個人アカウントを始めるまでの道のりも長かったり、
始めてからも撮影の時間や場所を探すのに手間取ったり、
静止画ではなくて動画全身だからよりリアルに視聴者にビジュアルが届いたり。笑笑
↑やだ！wwwwwwああああ(● ˃̶͈̀ロ˂̶͈́)੭ꠥ⁾⁾









そうだ、聞いて、
自分でtiktok撮影用スタンドを買ったりもしました。💸
もう元取れるくらいバズりたいよな！笑笑








そんなこんなで
山あり谷あり
色々あって今に至るのです。笑笑

自分からやりたいと始めたことだから、
こうして結果が出始めて嬉しかったです。

だからね、
あれやりたい、これやりたいに、繋がったり、
新しい可能性も秘めているので、
皆さんがワクワクしながらtiktok見守ってくれてるといいな🌙

tiktokに限らないけど。🌙🌱














⚪︎歌番組呼ばれなかったヒグチカッター

これはあの、showroomでもした「世知辛いぜ🍵」って話なのですが。笑笑

欅坂46さんの「世界には愛しかない」
を踊っているtiktokを撮影した裏話です。








この日、私は歌番組の収録に呼んでもらえなくて
かなりの空き時間を有していました。（ ; ; ）

私は残念だったけど、もちろん呼ばれたメンバーもいる。

しっかりメンバーをお見送りして、
さて、メンバーが戻ってくるまで、
かなりの空き時間です。









。。。よし、踊ろう！٩( ᐛ )و

加入前気分が落ち込んだ時は、
アイドルの曲を歌って踊ると元気になれたのです。

昔テレビで観てた乃木坂46さんの帯番組。
西野七瀬さんや齋藤飛鳥さんだって
呼ばれなかった過去があったの
私、観てました。









そんな、テレビを観ていた頃を思い出していたから、
ちょっと懐かしい「世界には愛しかない」を思いついたのかも知れません。








撮影して投稿したら、
欅坂46さんのファンの方々も反応してくれて、
私にとっても気に入ってる動画のひとつになりました。












⚪︎最後に
tiktokを通して、四条氏、手ごたえを感じました！

きっかけがあれば、
22/7のファンでない方からも
私に興味を持ってもらえる。

私には興味をもってもらえるだけの魅力があるんだ！
可能性があるんだ！って再認識しました。

また、
SNSだけでみると、
X(Twitter)とはまた違った良いところを沢山見つけました💪🏻
YOASOBIさんや、ゆら猫さんにも、
動画に反応してもらえたの嬉しかったなあ(*´-`)

なにより、
私のダンスや楽しそうに踊る表現を、
知ってもらえる場になりました。

tiktokには動画の種類が様々なのですが、
自分の「パフォーマンス」で再生回数が伸びた事実が嬉しかったです。

やりたいこと、得意なこと、求められてること、
その全部が重なる場所を見つけることができました。

そうだ！
計算中でカットされちゃったけど、
相田さんからメンバーへのお褒め言葉のコーナーでね。
まず一之瀬ちゃんからどうぞ！ってなって。

「一之瀬ちゃんはダンスがいいね！」って
開口一番に言ってもらえたの！嬉しかったです！！

小宮さんには、
ジェネレーションギャップで話分からなかった際、
気ィつかって愛想笑いしてくれるねって
言われました😂😂😂
愛想笑いバレてるぅwwww








おしまい。
四条月🌙



欢迎光临，
我是四条月是也( ´ ▽ ` )

















【今日内容】
⚪︎用数字结果来说你好
⚪︎说服力为零
⚪︎没被叫去上歌唱节目的樋口切割
⚪︎最后











⚪︎用数字结果来说你好
在TikTok的个人账号上，发生了一件开心的事。。。
竟然！！！
某个舞蹈视频，播放量突破了30万次！！！

👏🏻👏🏻👏🏻👏🏻👏🏻👏🏻👏🏻👏🏻（ ; ; ）
也就是发生了所谓的现代用语——“爆火”现象。









↑这就是我们所属事务所，Buzzwave的“Buzz”（爆火）哦😎✌🏻

每次一爆火，通知就响个不停，
让四条的手都忍不住地颤抖。(´・ω・)好可怕










其实在之前2023年4月，
开设个人账号后的第4个视频，
舞蹈视频也火过一次。

那个视频也快要达到15万次播放了🎉








当时那第4个TikTok视频播放量增长的时候，
虽然很开心，但我完全没有飘飘然。

如果不拿出下一个“结果”的话就没有意义，
或许是因为太认真了，当时我的表情可能很吓人。笑笑









结果，这次实现了超过上次爆火的第4个视频2倍以上、
突破30万次播放的
大火。。。！











能够抓住这个结果，
我打心底里感到高兴。
而且终于能稍微放松一点了( ´ ▽ ` )









之前一直紧绷着脸的原因是，

我很想用数字做出成绩。
拍摄每一个视频都很不容易。
以及我一直坚持说想开始做TikTok。

这些想法一直在脑子里转个不停。☁️













⚪︎说服力为零

关于我一直坚持说想开始做TikTok这件事。。。

其实我一直一直都在请求，
希望能让我饰演的一之濑萤酱也去挑战TikTok。











但是呢，要让萤酱动起来，
理所当然地比我动起来要难得多。

所以如果我不先做出成绩的话，是很难实现的。









虽然说“想做”很简单，
但以前的我是没有说服力的。

那是当然的啦( ＾∀＾)毕竟是新人。可没时间消沉哦ー♪









为了TikTok，得提早完成本职工作的Live练习，
在开始个人账号前的路也很漫长，
开始之后在寻找拍摄时间和地点上也费了不少功夫，
而且因为不是静止画而是全身视频，外形会更真实地传达给观众。笑笑
↑不要啊！wwwwww啊啊啊啊(● ˃̶͈̀ロ˂̶͈́)੭ꠥ⁾⁾









对啦，听我说，
我还自己买了TikTok拍摄用的支架。💸
好想火到能把本钱赚回来的程度啊！笑笑








就这样，
历经了各种起伏，
发生了许多事才有了现在。笑笑

因为是自己主动说想做的，
所以能像这样开始出成果真的很开心。

所以呢，
这也会连接到“想做那个”、“想做这个”，
其中也隐藏着新的可能性，
希望大家能怀着期待的心情关注我的TikTok呢🌙

虽然不仅仅限于TikTok就是啦。🌙🌱














⚪︎没被叫去上歌唱节目的樋口切割

这是那个，我在Showroom也提过的“世态炎凉呀🍵”的话题。笑笑

是拍摄跳欅坂46桑的《世界上只有爱》TikTok视频时的幕后故事。








这一天，我没能被叫去参加歌唱节目的录制，
空出了大把的时间。（ ; ; ）

虽然我很遗憾，但当然也有被叫去的成员。

好好地送走了成员们，
好了，在成员们回来之前，
是好长一段空闲时间。









。。。好，来跳舞吧！٩( ᐛ )و

在加入之前心情低落的时候，
唱跳偶像的歌曲就能让我恢复精神。

以前在电视上看的乃木坂46桑的常规节目。
就连西野七瀬桑和齋藤飛鳥桑也曾有过没被叫去录制节目的过去，
这些我都在节目里看过的。









或许是因为回想起了以前看电视的时光，
才突然想到了这首让人有些怀念的《世界上只有爱》吧。








拍摄并投稿之后，
欅坂46桑的粉丝们也给了很多回应，
这也成为了我非常喜欢的视频之一。












⚪︎最后
通过TikTok，四条月感受到了收获！

只要有契机，
即便是非22/7粉丝的人，
也会对我产生兴趣。

我再次认识到，
我有足以让人产生兴趣的魅力！
是有可能性的！

另外，
仅仅从社交媒体来看，
我也发现了许多与X(Twitter)不同的优点💪🏻
YOASOBI桑，还有ゆら猫桑也在视频里给了互动，
真的好开心啊(*´-`)

最重要的是，
这里成为了能让大家了解我的舞蹈、
以及我享受舞蹈的各种表现的场所。

虽然TikTok上的视频种类繁多，
但通过自己的“表现力”让播放量增长，这个事实让我非常开心。

想做的事、擅长的事、被期待的事，
我终于找到了这三者重合的地方。

对啦！
虽然在《计算中》里被剪掉了，
但在相田桑表扬成员的环节里。
最开始说请一之濑酱先来！

“一之濑酱的舞蹈很棒呢！”
他第一句话就这么夸我了！真的好开心！！

小宫桑则对我说，
在遇到代沟听不懂话的时候，
我会很客气地在那里陪笑呢😂😂😂
陪笑被识破了呀wwww








结束。
