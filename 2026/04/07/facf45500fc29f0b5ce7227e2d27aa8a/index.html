22/7 LIVE 「15」ありがとうございました。22/7 3期生の南伊織です。


ライブの前日、1人でダンスの復習をしていました。何だかそわそわしてしまって。その後気合いを入れるためにメロンソーダを飲みました。多分飲み物の中で1番好きだと思います。そんなギリギリまで確認と復習をした15のライブはいかがでしたか。今の南にできる精一杯をみなさんにぶつけに行きました。南の全力が伝わっていると良いな。みなさんに自分の届けたい想いが伝わっていたら嬉しいです。今回のライブの盛り上がりポイントの1つに、君とどれくらい会わずにいられるか？のところでサインボールを投げるというのがありました。南の愛のボールです。受け取ってくださった方は大切にしてくださると嬉しいです。よろしくお願いします。今回のライブでは、先輩方から学ぶことが多かったと感じています。今までは、先輩方を舞台袖や観客席、映像で見ていることがほとんどだったので、近くで一緒に踊ったり歌ったりすることで、より多くのものを得られた気がしています。一体どこまで行けば先輩方に追いつけるのだろうか。踊り方、表情管理、魅せ方など本当に全ての動作に研ぎ澄まされたものを感じました。近くにいると、その美しさに息を呑み、目を奪われます。自分もそう思ってもらえるように頑張らないとですね。今回の経験を活かして、みなさんやこれからナナニジと出会ってくれるであろう方々の心を動かすことができるパフォーマンスを目指していきます。心から尊敬できる先輩方と同じステージに立つことができて、幸せを噛み締めています。天城さんは、カメラへのファンサや動きの躍動感がかっこよくてダンスのキレなどメリハリや個性を出してダンスで魅せるということなどを学ばせていただきました。また、MCが面白いなどのギャップも魅力的で、本当に全てを尊敬しています。河瀬さんは、いつどこにいても目を惹くオーラや華やかさがあり、本当に素敵です。そして、指先がとても美しいですし、目線の使い方なども綺麗で気を抜くと好きになってしまいます。動きに優雅さがあり、ポーズなどの形が美しいので、自分も人からの見られ方を考えるようになりました。相川さんは、ダイナミックな動きと体全体を大きく使っていると感じるダンスだと思っていて、同じパートを歌わせてもらう時はどうしたら相川さんのように踊れるかずっと観察していました。あと、相川さんの22on7のラップが大好きです。麻丘さんは、ものすごく視野が広くて3期生のことをたくさん見てくださっていました。ダンス細かい振りやタイミングを教えてくださったり立ち位置なども見てくださっていて、そんなところを見ていたからこそ、麻丘さんの繊細で緻密な表現に惹き込まれるだと思いました。椎名さんは、本当に歌声と表情が素敵だと思っています。特に切なさなどの感情や淡さを感じる唯一無二なところが大好きで魅力的な椎名さんを見て、自分も表現力をもっと身につけたいと思いました。月城さんは、息をするように滑らかなダンスが本当に素敵で、その流れるような美しさが静止するダンスでどちらもより魅力になると教えていただきました。振付師の方に止まる瞬間をちゃんとできるようにと教えていただいていたので、月城さんのすごさを本当に間近で感じることができて嬉しかったです。望月さんは、最初から最後まで笑顔で360°どこから見ても可愛くて素敵なアイドルでとても尊敬しています。望月さんの歌声や声量で周りの雰囲気が変わる感じやダンスの正確さなどに思わず目で追いたくなってしまう魅力があり、自分もそうなれるように頑張りたいと思いました。こんな素敵な先輩方がいる22/7に南のことを入れてくださって、ありがとうございます。アイドルというものは応援してくださる方がいて成立するものだとつくづく思いましたし、今の自分にもみなさんのような存在がいてくださることがさいっっっこうに幸せです。本当にありがとうございます。今を精一杯生きて、南の生き様、南のアイドル人生をみなさんに見てもらおうと思います。南はまだまだ髪を振り乱して、汗をかいて前に進み続けます。見届けてね。実はそう思う理由がありまして、多分15のリハ中に1番指摘されたことが多かったのって自分なんですよ。ダンスのスピードには追いつけず、振りは間違えるばかりで。毎日練習しているはずなのに、どうしてこんなにできないんだろうって思いながら、涙を堪えて練習していました。そこで南のハートは粉々に砕け散りましたが、粉々になったのなら新しいハートの形に作り直せば良いと思って。今は形は歪かもしれないけど、はみ出した部分は少しずつ削れば良いし、不完全な方がきっと面白いから。自分が1番できないことくらい合格した時からわかってたし、俯いてる時間がもったいない！
そして、たくさんのお花もありがとうございます。本当にお花大好きです。一緒に撮ることができなくてすみません、時間が足りなくて、すべてのお花をお写真に収めることができなかったかもしれないです、本当にすみません。でも、ちゃんとお花の美しさは心に刻んでいます。お花を出してくださったみなさん、本当にありがとうございます。


また、オンライン特典会ありがとうございました。ライブの翌日だったこともあり、たくさんの方がライブの感想を聞かせてくださって、嬉しかったです。命の続きや嫌われるということのセリフを褒めてくださった方やシャンプーの匂いがしたの笑顔を褒めてくださった方もいらっしゃいました。本当にありがとうございます。みなさんはどの曲を歌って踊る南が良かったですか。ぜひ教えてください。自分自身は謎の力は踊っていて、歌っていて楽しかったです。あまり3期生ではやってこなかった元気や明るさを感じる曲ということもあって新鮮でした。実は、謎の力はオーディション中に1番聴いていた曲で、3次審査の時に書道で文字を書いて見せようと決めた時に、謎の力の歌詞を書いていたんです。3次審査当日のギリギリまで悩んで結局見せずに終わりました。それくらい謎の力も大好きな曲です。特典会に話を戻しますね。ライブに来てくださって、特典会にも来てくださった方は本当に大変忙しかったと思います。ありがとうございます。ゆっくり休んでください、お身体は大切に、無理しないで、みなさんが元気でいてくださることが一番ですから。いつだってどこだってみなさんのことを考えて、物思いに耽っています。だから、少しくらいは南のことも考えてくださると嬉しいな、なんて。今日は、前回のオンライン特典会でお会いできなかった方などにも会えたり、お久しぶりに会いに来てくださった方にも会えたり、いつも会いに来てくださる方にも会えたり、初めましての方にもお会いできたりして幸せでした。本当にありがとうございます。ごめんだけど、もうみなさんに会いたい！！！！！！みなさんのこと大好きすぎて、どうにかなりそう､､､みなさんロスです。


椎名さんにバームクーヘンをいただきました！！！！！！食べるのもったいなかったけど、バームクーヘンの魔力に負けてしまった。本当に美味しかったです。ありがとうございます。いっつも大好きです。LOVEです。


みさきちがお菓子とかあったかくなるものをくれてお手紙もくれました。本人は出すのを嫌がっているので、ここで自分もお手紙の返答をしたいと思います。本番お疲れさまでした。みさきちはいっつも3期生にダンスを教えてくれたり、わからないことはなんでも教えてくれてありがとう。みさきちは3期生のパフォーマンスをすごく引き上げてくれていて功労者です。みさきちのソロかっこ良かった！


ねむいおり、みなさんゆっくり休んでくださいね、おやすみなみ。

あ、南のはずれ文字どうでしたか。

感谢大家前来观看 22/7 LIVE 「15」。我是 22/7 3期生的南伊织。


Live 的前一天，我一个人在复习舞蹈。总觉得心里七上八下的。之后为了给自己鼓劲，我喝了哈密瓜汽水。这大概是我最喜欢的饮料了。就是这样直到最后一刻都在确认和复习的「15」Live，大家觉得怎么样呢？我将现在南所能做到的全力以赴都展现给了大家。如果南的全力能传达给大家就好了呀。如果我想要传达的心意能让大家感受到，我会很开心的。这次 Live 的一个高潮环节，是在唱《君とどれくらい会わずにいられるか？》的时候扔签名球。那是南的爱心球哦。收到的朋友如果能好好珍惜它，我会很开心的。拜托大家啦。在这次 Live 中，我感觉从前辈们身上学到了很多。至今为止，我大多都是在舞台侧翼、观众席或是通过影像看着前辈们，而这次能在近处一起唱歌跳舞，感觉收获了更多的东西。究竟要走到哪里，才能追上前辈们的脚步呢？舞蹈的方式、表情管理、展示自己的方法等等，真的感觉她们的每一个动作都经过了千锤百炼。在她们身边，会被那份美丽所震撼，夺去目光。我也必须努力，让自己也能成为别人眼中那样的存在。我会活用这次的经验，为了能给各位、以及今后将要与娜娜尼基相遇的人们带来触动心灵的表演而努力。能和发自内心尊敬的前辈们站在同一个舞台上，我细细品味着这份幸福。天城前辈，她对摄像机的饭撒和富有动感的动作非常帅气，舞蹈的利落感等张弛有度和个性的展现，让我学到了如何用舞蹈来吸引观众。而且，MC时非常有趣之类的反差萌也很有魅力，真的所有的一切都让我尊敬。河瀬前辈，无论何时何地都散发着引人注目的气场和华丽感，真的非常出色。而且，她的指尖非常优美，视线的运用也很漂亮，一不留神就会喜欢上她。她的动作中带着优雅，姿势的定格之类的也非常优美，让我自己也开始思考别人眼中的我是怎样的。相川前辈，我认为她的舞蹈动作大开大合，感觉整个身体都在大幅度地活动，当我和她唱同一个部分时，我一直在观察要怎么做才能像相川前辈那样跳舞。还有，我超喜欢相川前辈在 22on7 里的 rap。麻丘前辈，她的视野非常开阔，一直非常关照我们 3 期生。她会教我们舞蹈中细微的动作和时机，也会帮我们看站位，正因为她注意到了这些细节，我才会被麻丘前辈那纤细而又致密的表现力所吸引。椎名前辈，我真的觉得她的歌声和表情非常出色。特别是她那份能让人感受到悲伤等情感和一丝淡然的独一无二之处，我最喜欢了，看着充满魅力的椎名前辈，我也想更加提升自己的表现力。月城前辈，她那如呼吸般流畅的舞蹈真的太棒了，她教会了我，那种行云流水般的美感和静止的舞蹈动作相结合，两者都能变得更具魅力。因为舞蹈老师也教过我要把定格的瞬间做好，所以能如此近距离地感受到月城前辈的厉害之处，我真的非常开心。りの，从头到尾都带着笑容，360°无死角地可爱，是一位非常出色的偶像，我非常尊敬她。りの的歌声和声量能改变周围的气氛，舞蹈的精准度等等，都有一种让人不自觉地追随她目光的魅力，我也想为了成为那样的人而努力。能让我加入有这么多出色前辈的 22/7，真的非常感谢。我深切地感受到，偶像是有了支持我们的人才得以成立的职业，现在的我能有大家这样的存在，真的是太太太太幸福了。真的非常感谢。我会拼尽全力活在当下，希望大家能见证南的生活方式，南的偶像人生。南还会继续甩动头发、挥洒汗水，不断前进的。请一定要见证哦。其实之所以会这么想，是有原因的。在「15」的排练中，被指点次数最多的可能就是我了。跟不上舞蹈的速度，动作也总是出错。明明每天都在练习，为什么就是做不好呢，我一边这样想着，一边忍着眼泪练习。虽然南的心在那里碎得粉粉的，但我想，既然碎了，那就重新塑造一个新形状的心好了。现在这颗心的形状也许还很扭曲，但多出来的部分可以一点点削掉，而且不完美的样子说不定才更有趣。我从合格的时候就知道自己是做得最差的那个了，所以没时间低着头浪费！


然后，也非常感谢大家送来的许多花篮。我真的超级喜欢花。没能和花篮一起合影真的很抱歉，因为时间不够，可能没能把所有的花篮都拍进照片里，真的非常抱歉。但是，花儿们的美丽都已经深深地刻在我的心里了。送花给我的各位，真的非常感谢。



另外，线上特典会也非常感谢大家。因为是 Live 的第二天，很多人都跟我分享了 Live 的感想，我非常开心。有夸奖我在《命の続き》和《嫌われるということ》中台词的朋友，也有夸奖我在《シャンプーの匂いがした》中笑容的朋友。真的非常感谢。大家觉得南唱跳哪首歌的时候最棒呢？请一定要告诉我呀。我自己的话，跳《謎の力》的时候，唱歌跳舞都非常开心。这首歌很有活力和开朗的感觉，是我们 3 期生不怎么尝试过的曲风，所以感觉很新鲜。其实，《謎の力》是我在甄选期间听得最多的一首歌，在第 3 次审查时，我决定用书法写字来展示，当时写的就是《謎の力》的歌词。结果一直纠结到 3 次审查当天的最后一刻，最后还是没有展示出来。就是这么一首我也非常喜欢的歌。说回特典会。既来看了 Live，又来参加特典会的朋友们，真的非常辛苦。谢谢你们。请好好休息，保重身体，不要勉强，大家健健康康的就是最重要的。无论何时何地，我都在想着大家，沉浸在思念中。所以，如果大家也能稍微想一想南的话，我会很开心的，嘿嘿。今天，能见到上次线上特典会没能见到的朋友，能见到好久不见的朋友，也能见到一直来见我的朋友，还能见到初次见面的朋友，我非常幸福。真的非常感谢。对不起啦，但我已经好想见大家了！！！！！！我太喜欢大家了，感觉要不行了､､､我现在得了“想念大家”综合症。


椎名前辈送了我年轮蛋糕！！！！！！虽然觉得吃掉太可惜了，但还是输给了年轮蛋糕的魔力。真的非常好吃。谢谢您。我一直都最喜欢您了。是 LOVE。



みさきち给了我零食和一些暖宝宝之类的东西，还给了我一封信。因为她本人不想公开，所以我就在这里回复一下信吧。正式演出辛苦啦。みさきち总是教 3 期生跳舞，有什么不懂的都会教我们，谢谢你。みさきち极大地提升了 3 期生的表演水平，是我们的功臣。みさきち的 solo 好帅！


困困的伊织，大家要好好休息哦，晚安南。
