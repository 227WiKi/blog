大変大変遅くなりました💦

後、2日で定期公演ファイナル!!

遅くなりましたが、
一緒に5月定期公演を振り返ってください！🙇🏻‍♀️´-



【5月定期公演】

1日目は、いこちゃん＆まにゃの影ナレから

2日目は、あべせんと木村。あ、違う。 
みれたんと南ちゃんの影ナレから始まりました📖  ̖́-

Overtureから、
韋駄天娘➡️地下鉄抵抗主義➡️
嫌われるということ➡️僕らの環境

初手4曲という、今までの定期公演とは違う「15」を経ての私たち3期のスタート✨️

1.2月の定期公演はずっと「命の続き」からのスタートだったので、
Overture緊張しましたぁ:( ;´꒳`;):

でも、韋駄天娘から全力で駆け抜けた4曲です!!

嫌われるということでは、みんなで話し合って、新しい表現をしてみました!!
気づいた方は3期の表情を見ている方ですね！👀💕︎(多分)
今までは、怒りのような、どうにもできない心と未来に抗うような荒々しい表現でしたが、
5月定期公演では、そんな状況も受け入れて、自分の道を進むというような…吹っ切れた表現をしてみました!!
伝わりましたかね…( 'ω'   )

そして僕らの環境!!

有難いことに、ソロダンスを務めました🙇🏻‍♀️🌷
実は1.2日目でそれぞれ違うダンスを披露しました!!
気づいた方も居たみたいで嬉しいです😭🫶
直前まで変えるか迷っていてのですが、今回は日替わり楽曲がなかったので、少しでも変化を感じて欲しいと思い変えてみました…

変えることを決めた時に、
南ちゃんには移動の関係で伝えたのですが、
驚かれました。ごめん南。( 'ᢦ' )

ありすちゃんとかも変えたよね?!かっこよかった✨️って言われて心の中ではニンマリ(  ◜ω◝ )三雲氏。へへ。


そして、定期公演といえば…

✨️ソロコーナー✨️

1日目のみれたん!!

念願の日本舞踊をやっと見れて嬉しさと感動で一杯でした😭

本当に綺麗でずっと見とれてました(´꒳`)

身長が高いからこそ映える着物姿は目に焼き付いております👀👀👀

そして、独白。

もうね、泣いちゃいそうでした。心の中では号泣よ。号泣。

みれたんの知らない一面をここで見れた気がして…言葉一つ一つに思いがぎゅっと詰まった独白。この言葉を聴いて、これからもみれたんとみんなで一緒に頑張っていこうと思いました🥹

雲晴も最高でした😭👏✨
歌声もダンスも本当に最高で感動もの😭

指先まで意識したダンスに、優しい歌声。

みれたんの良いところがぎゅっと詰まったソロコーナーでした！

ステージから戻って来た時の笑顔は本当にキラキラしてて、こっちまで貰い泣きじゃなくて、貰い笑顔になりました😆




2日目のソロコーナーは、私。

独白では、ありすちゃんと南ちゃんにお手伝い頂き、人形劇で行いました🧸

ドキュメンタリーに｢手汗気まずい」が使われてて笑っちゃいました

本番も出る前に南ちゃん手汗拭いてとお願いをしました。＿__＿○＿(土下寝)

独白の内容は、ずっと私が悩み続けていることでもあり、解決しようにもできないと思っていた時に、みさきから頂いた言葉たちです。

まだ森で迷子になっているけど、

誰よりもゆっくりな歩みだけど、

それでも進んでいきます。

これからも、暖かく見守ってくれると嬉しいです。


向日葵色の光の海を忘れられません。

とってもとっても綺麗でした。



でも、嬉しかった反面

1色だけの景色は少し寂しかったです。

ステージから見る景色は、大好きなみんなの色がいいなと。

その方が、勇気が貰える気がするから。

もちろん、皆さんの優しい空気のおかげでやりきる事が出来ました。

本当にありがとうございました。

御礼が遅くなってしまい、大変申し訳ないです💦

本当に、本当にありがとうございました。

と言って頂いたこの子にしました( ¨̮ )

朗読劇📖 𓂃𓈒 𓂂𓏸

この2人の空気感って不思議だよね。

普段は姉妹より兄弟という表現が合いそうなくらい何でも言い合えて、この2人の掛け合いには誰も入れないくらい独特なの。

けど、それとはまた違う2人の空気感。

物語の中ではあるんだけど、兄弟から同級生になって、同じ学校だったら…なんて想像してました🥴

この2人のキャラクターの違いが個性的で好きですね。

いこちゃんの空気を含んだ柔らかい声質に、

南ちゃんのコミカルだけど、その中に強さもある声質。

温度差も含めて面白さが勝ってしまう🤭

そして、最後の告白シーン💌

ずーーーーっと照れてたの！
リハから!!ずーーーっと！
こっちが照れちゃうくらい!!

でも本番はきっちりと決めていくのがこの2人。

流石です!!最高の朗読劇でした。



2日目はまな&ありすのうさぎ小屋。

このふたりの空気感も好き。

沢山お話してくれるまなの話をずっと頷きながら聴いているありすちゃん。

この日もそう。女の子の話を真剣に聞いてくれる僕。

この2人は逆にいつもの雰囲気と近しいものを感じた。

まなはもう少しテンション高いけどね( 'ᢦ' )

でも、声質も本当にピッタリだし、

2人ともお人形さんと一緒でうさぎみたいだし🐇

本当にピッタリな朗読劇でした。

本当に素敵な時間だったよね。

**博客**


真的真的非常抱歉来晚了💦

再过2天就是定期公演的 Final 了!!

虽然有点晚了，
但还请大家和我一起回顾一下5月的定期公演吧！🙇🏻‍♀️´-



【5月定期公演】

第一天，是从 iko酱&茉喵的后台广播开始的

第二天，是从阿部老师和木村。啊，不对。 
是从 Miretan 和 南酱的后台广播开始的📖  ̖́-

从 Overture 开始，
接着是韦驮天娘➡️地下铁抵抗主义➡️
被讨厌这件事➡️我们的环境

开场就是四连曲，这是和至今为止的定期公演都不一样的、经历了「15」之后的我们三期生的新起点✨️

1、2月的定期公演一直都是从「生命的延续」开始的，
所以 Overture 的时候好紧张啊:( ;´꒳`;):

不过，从韦驮天娘开始，我们全力以赴地冲过了这4首歌!!

在《被讨厌这件事》这首歌里，大家一起讨论后，尝试了全新的表现方式！！
注意到的人，一定是平时有在好好看我们三期生表情的人吧！👀💕︎(大概)
至今为止，我们的表演都是像愤怒一样，带着一种对抗无可奈何的心情与未来的粗犷感，
但在5月的定期公演上，我们试着表现出一种接受了这种状况，并继续走自己的路的…释然的感觉！！
大家有感受到吗…( 'ω'   )

然后是《我们的环境》!!

非常荣幸地，我担任了 Solo Dance 的部分🙇🏻‍♀️🌷
其实第一天和第二天，我表演了完全不同的两段舞蹈！！
好像也有人注意到了，好开心😭🫶
直到上场前我还在犹豫要不要改，但这次没有每日替换的歌曲，所以想着至少让大家感受到一点点变化，就试着改了…

决定要改的时候，
因为走位的关系我告诉了南酱，
结果她吓了一跳。对不起啦南。( 'ᢦ' )

Alice酱她们也对我说“你也改了对吧？！超帅的✨️”，我心里可是美滋滋的(  ◜ω◝ )三云氏。嘿嘿。


然后，说到定期公演的话…

✨️Solo环节✨️

第一天是 Miretan!!

终于看到了心心念念的日本舞踊，我心里充满了开心和感动😭

真的太美了，我一直都看入迷了(´꒳`)

正因为她身材高挑，穿和服的样子才格外亮眼，这画面已经深深地烙印在我的脑海里了👀👀👀

然后是独白。

真是的，我差点就哭出来了。心里已经在号啕大哭了哦。号啕大哭。

感觉在这里看到了 Miretan 不为人知的一面…她的独白，字字句句都凝聚着情感。听了这些话，我下定决心今后也要和 Miretan、和大家一起继续努力🥹

《云晴》也超级棒😭👏✨
歌声和舞蹈都真的太棒了，让人感动😭

舞蹈动作细致到指尖，歌声又那么温柔。

是一个把 Miretan 的优点都浓缩在一起的 Solo 环节！

她从舞台上回来时的笑容真的闪闪发光，让我也不由自主地跟着她一起笑了起来，而不是被感动到哭😆




第二天的 Solo 环节，是我。

在独白部分，我请了 Alice酱和南酱来帮忙，用人偶剧的形式进行了表演🧸

纪录片里把我那句“手汗好尴尬”给用进去了，笑死我了

正式上场前我也拜托南酱“帮我擦擦手汗”。＿__＿○＿(五体投地)

虽然现在我还在森林里迷着路，
